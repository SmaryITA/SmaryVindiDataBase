# SmaryVindiDataBase

A complete database for Vindictus with all the necessary tables for Raid, Redeemer, Space-Time, Accessories, Uaithne/Eriu Gear, Tuning and Surplus.

## Content

- Complete tables for every type of game content
- Easy-to-use HTML pages
- Responsive design (works well on mobile)

## How to use

1. Download or clone the repository
2. Open `index.html` with any browser
3. Navigate through the tables using the links

## Technologies

- HTML
- CSS
- JavaScript
